Understanding Structure of HTML

- HTML controls the structure of the page
- CSS Controls the style of the page.
- HTML is written in a tree like structure, which the browser reads to form the page
- This tree like structure is sometimes referred to as "DOM"
- Each element has an opening and a closing tag
- Each successive element is embedded further into the webpage. The more indented it is the deeper it is embedded

The Importance of Avoiding Repetition

- It is important to always avoid repetition when possible
- This helps to keep the code much shorter and cleaner
- Helps to prevent minor errors made through unnecessary retyping and configuring of stylings
- Creates a more reader friendly version of the code
- Makes it more fun to work on, instead of typing lines and lines of unnecessary code!
